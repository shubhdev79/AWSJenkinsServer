[  Terraform installation on Linux/CentOS: ] 

sudo curl -O https://releases.hashicorp.com/terraform/0.12.20/terraform_0.12.20_linux_amd64.zip

sudo yum install -y unzip

sudo unzip terraform_0.12.20_linux_amd64.zip -d /usr/local/bin

sudo unzip terraform_0.12.20_linux_amd64.zip -d /usr/sbin - To Install in ROOT.
